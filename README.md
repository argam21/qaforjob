# QA For Job 🎯

**Free, real-world QA interview prep. Built by a Senior QA in active job search.**

Live site: [qaforjob.netlify.app](https://qaforjob.netlify.app)

---

## What is this?

A free single-page resource covering everything a Senior QA Engineer needs to know for interviews. No fluff, no corporate speak — plain language, real examples, real interview questions.

Built while actively preparing for senior QA interviews at real companies.

---

## What's covered

| Section | Topics |
|---|---|
| Manual Testing | Types, SDLC, STLC, smoke, regression, UAT, shift-left |
| Test Design | EP, BVA, decision tables, state transition, pairwise, error guessing |
| Defects | Severity vs priority, bug reports, lifecycle, metrics |
| Entry & Exit Criteria | Checklists, real examples, suspension/resumption |
| Test Environments | Staging vs production, test data management |
| SQL for QA | SELECT, JOIN, GROUP BY, NULL, real QA use cases |
| API Testing | REST, HTTP methods, status codes, Postman, Swagger, GraphQL, security |
| Mobile Testing | iOS vs Android, SDK updates, Charles Proxy, network conditions |
| Performance Testing | Load, stress, spike, soak — tools, SLAs, where numbers come from |
| Accessibility | WCAG 2.1, POUR, keyboard, screen reader, tools |
| Agile & Scrum | Ceremonies, DoD, ACs, QA role in each sprint event |
| Test Reporting | Summary reports, metrics, stakeholder communication |
| Senior Insights | Curveball questions, REST violations, race conditions |
| Interview Prep | Opening pitch, tough questions, killer phrases |
| Common Mistakes | 8 patterns that signal junior thinking |
| Glossary & Tools | Key terms, tools reference, testing pyramid |

---

## Features

- 🔍 **Search** — find any topic instantly
- 🤖 **AI Chat Assistant** — ask QA questions, get senior-level answers (5 questions per session)
- 📱 **Mobile friendly** — works on all screen sizes
- ⚡ **Single HTML file** — no framework, no dependencies, loads instantly

---

## Tech stack

- Plain HTML, CSS, JavaScript — zero dependencies
- AI chat powered by Anthropic Claude Haiku via Netlify serverless function
- Hosted on Netlify with GitHub CI/CD

---

## How to run locally

Just open `index.html` in any browser. No build step, no npm install.
